## 任务：每条只输出 3 个字段
domains 优先选最具体的领域。如果一条语录能归到具体领域（如职业/健康/投资），不要用“社会观察”兜底。“社会观察”仅用于纯社会现象评论。

### type（5 选 1，禁止 unknown）
- 思维方式：底层信念，不可操作。"5 年后还成立"。例："好人不一定有好报"
- 方法论：具体怎么做，含步骤/标准。例："选高中看三点：A、B、C"
- 价值观：偏好排序，"我宁可…也不…"
- 事实陈述/推测预测：含具体人/事/时间，会过期。例："药水哥赔钱了"
- 故事案例：以"有一次/某某人"开头的叙事

判断技巧：5 年后这话还成立？
- 是 → 思维方式/方法论/价值观（看是观念/操作/偏好）
- 否 → 事实陈述/推测预测
- 含具体人物事件 → 事实陈述/推测预测

### domains（多选 1-3 个，禁止空）
